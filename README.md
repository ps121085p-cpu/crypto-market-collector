# Crypto Market Data Collector

## 📌 Description

This project collects real-time cryptocurrency data from CoinGecko API and stores it in PostgreSQL database and CSV file.

## ⚙️ Technologies

* Python
* requests
* PostgreSQL
* psycopg2

## 🚀 Features

* Fetch top 10 cryptocurrencies from API
* Store data in PostgreSQL
* Export data to CSV file
* Simple and clean architecture
* Error handling implemented

## ▶️ How to run

1. Install dependencies:
   pip install -r requirements.txt

2. Set your database credentials in main.py

3. Run script:
   python main.py

## 📊 Output

* Database table: `crypto_market`
* CSV file: `crypto_market.csv`

## 💡 Use cases

* Data collection
* Market monitoring
* Backend data pipelines
